# Heavy Agent Profile

Opt-in profile for expensive, broad, or special-purpose tools.

Enable these only when the task benefits from them:

- Google Workspace write tools;
- presentations, spreadsheets, document rendering, and image generation;
- browser tracing and full browser automation;
- release automation;
- broad tech-debt audits and repo-wide architecture reviews;
- oncall handoff and large Slack/report synthesis;
- domain-specific skills such as media analysis, Ghidra, OpenSCAD, Sentry, or
  public-transport helpers.

These tools are useful, but they increase context surface, permissions, startup
noise, or dependency weight. They should be easy to install, easy to discover,
and absent from the default context until needed.
